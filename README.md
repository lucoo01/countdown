# countdown (死亡倒计时)

这是个桌面程序,使用Electron开发.

## 使用方法
* 到软件发布页面下载安装包安装: https://github.com/lucoo01/countdown/releases
* 源码安装

```bash
# 将代码克隆到本地
git clone https://github.com/lucoo01/countdown.git
# 切换到软件目录
cd countdown
# 安装
npm install
# 运行软件
npm start
```
